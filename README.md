# AlgorithmX Platform

A high-performance template for building data-driven applications with a React (Vite) frontend and Node.js (Express) backend.

## Tech Stack

### Frontend
- **Framework**: [React](https://reactjs.org/) + [Vite](https://vitejs.dev/)
- **Language**: TypeScript
- **Styling**: Vanilla CSS (Premium Glassmorphism Design System)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)

### Backend
- **Framework**: [Express.js](https://expressjs.com/)
- **Language**: TypeScript
- **Runtime**: Node.js

## Getting Started

### 1. Install Dependencies
```bash
# In backend/
cd backend && npm install

# In frontend/
cd frontend && npm install
```

### 2. Run Development Servers
```bash
# Start Backend (on http://localhost:3001)
cd backend && npm run dev

# Start Frontend (on http://localhost:5173)
cd frontend && npm run dev
```

## Features
- ✨ **Premium UI**: Custom-built design system with a high-end, futuristic aesthetic.
- 🚀 **Glassmorphism**: Beautifully blurred cards and interactive elements.
- 📡 **Integrated API**: Pre-configured proxy in Vite to communicate with the backend.
- ⚡ **Auto-Health Check**: Live system status indicator on the landing page.
- 🧪 **Scalable Design**: Modular folder structure ready for production deployment.
